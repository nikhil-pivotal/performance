# Performance

## Overall Process

Performance measurement can be broken down into a three stage process

 - Measurement
    - Calculation of returns, benchmarks and peer groups
 - Attribution
    - Return Attribution, Risk Analysis
 - Evaluation
    - Presentation of results, feedback and control
